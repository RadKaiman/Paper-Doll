# Paper Doll

# Описание
Игра-симулятор нанесения макияжа

# Технический стек
Unity 6
Zenject - Dependency Injection
UniRx - Реактивное программирование
DOTween - Анимации

# Основные механики
1. Крем
- Нажатие на крем => рука движется к предмету
- Перетаскивание к лицу => анимация нанесения
- Удаление прыщей с лица
- Автоматический возврат предмета

2. Помада
- Выбор помады => рука берет предмет
- Анимация краски губ (движения влево-вправо)
- Смена спрайта губ на соответствующий цвет

3. Тени
- Выбор цвета в палетке => рука берет кисточку
- Окрашивание кисточки выбранным цветом
- Анимация нанесения теней (широкие движения)
- Смена спрайта теней над глазами

4. Стирание макияжа
- Нажатие на голубой спонжик => мгновенное удаление всего макияжа

# Архитектура
MVC + Zenject

## Основные технические моменты реализации

### Архитектурные решения
1. Использование MVC + Zenject для четкого разделения ответственности
2. Реактивное программирование через UniRx для автоматического обновления
3. Все анимации реализованы через DOTween для плавности и контролируемости

### Работа с UI
1. Drag & Drop через интерфейсы IBeginDragHandler, IDragHandler, IEndDragHandler
2. Преобразование экранных координат в локальные для определения зоны лица
3. Управление картинками через включение/выключение

### Система предметов
1. ScriptableObject для хранения данных о макияже
2. Отдельные View для разных типов предметов
3. Состояния руки для каждого этапа взаимодействия

### Оптимизация
1. Отписка через CompositeDisposable
2. Инъекция зависимостей через Construct для MonoBehaviour
3. Использование ReactiveProperty для реактивных данных
